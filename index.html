1. ¿Cuál es la diferencia principal entre position relative y position absolute?

La diferencia es que al position absolute se le debe poner los valores para especificar en que parte de la caja se colocara el elemento

2. ¿Qué valor de position se utiliza por defecto para los elementos HTML?

Los valores por defecto que se utiliza es: top-bottom-left-right

3. ¿Qué propiedad se utiliza para establecer la capa de apilamiento de elementos superpuestos?

La propiedad que se utiliza es z-index

4. ¿Cuándo se utiliza position fixed en CSS?

Se utiliza cuando queremos dejar un elemento fijo en una parte de la página

5. ¿Cómo se comporta un elemento con position sticky al desplazarse?

Un elemento con position sticky se comporta de manera pegajosa, pudiendo arrastrar el elemnto por la página

6. ¿Cuál es el valor de position adecuado para un elemento que se desea que siga el flujo normal del documento?

El valor que se utiliza es "position:static"

7. ¿Cuál es la forma más común de mover un elemento con position relative?

La forma más común de mover un elemento con relative es con los valores top-bottom-left-right

8. ¿En qué casos se puede utilizar z-index para controlar el orden de apilamiento de elementos?

Los casos en los que se puede utilizar z-index son para apilar cajas
